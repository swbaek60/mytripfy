# 다음 달에 한글 번역 나머지 하기

**이번 달 결과:** ko 183건 완료 → 다음 달에는 **183번째부터** 이어서 하면 됩니다.

---

## 다음 달 할 일 (한글만 이어서)

### 1. .env.local 수정

`.env.local`에서 **CHALLENGE_OFFSET**을 **183**으로 넣습니다.

```env
LANGUAGES=ko
MAX_CHARS=500000
CHALLENGE_OFFSET=183
```

### 2. 시드 한 번 실행

```bash
npm run seed:translations
```

### 3. 끝나면

터미널에 **다음 CHALLENGE_OFFSET=숫자**가 나옵니다. 그 숫자를 메모해 두고, **그다음 달**에 같은 방식으로 `.env.local`의 CHALLENGE_OFFSET만 그 숫자로 바꾼 뒤 다시 실행하면 됩니다.

---

## 참고 (이번 달 완료 건수)

| 언어 | 이번 달 완료 | 다음 달 CHALLENGE_OFFSET |
|------|----------------|---------------------------|
| ko   | 183건          | **183** (한글 나머지 할 때 사용) |
| ja   | 817건          | 817 (일본어 나머지 할 때) |
| zh   | 817건          | 817 (중국어 나머지 할 때) |

한글(ko)만 이어서 하려면 **LANGUAGES=ko**, **CHALLENGE_OFFSET=183** 만 넣고 실행하면 됩니다.
