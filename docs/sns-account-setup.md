# 인스타그램·페이스북 계정 만들기 (수아 / 이든)

캠페인은 **오늘**부터 시작합니다. 수아는 **한국**, 이든은 **미국**에서 각각 출발합니다.

---

## 1. 계정 구성

| 캐릭터 | 인스타그램 계정 | 페이스북 페이지 | 출발 국가 |
|--------|-----------------|-----------------|-----------|
| **수아 (Sua)** | 1개 (비즈니스/크리에이터 권장) | 1개 페이지 | 🇰🇷 한국 |
| **이든 (Ethan)** | 1개 (비즈니스/크리에이터 권장) | 1개 페이지 | 🇺🇸 미국 |

- 인스타그램에서 **게시물 올리기** → 연결된 페이스북 페이지에 **자동 공유**되도록 설정합니다.

---

## 2. 인스타그램 계정 만들기

### 2-1. 수아 계정

1. **인스타그램 앱** 또는 [instagram.com](https://www.instagram.com) 접속
2. **회원가입** → 새 이메일·비밀번호로 가입 (수아 전용 이메일)
3. **사용자 이름** 예시: `sua.100countries` / `sua_mytripfy` / `travelwithsua` (원하는 이름으로)
4. **프로필 설정**
   - **프로필 사진:** 수아 대표 이미지 (예: `sua_captivating_sexy.png`)
   - **이름:** `Sua | 100 Countries`
   - **소개문:** 아래 복사해서 사용

```
✨ 1년에 100개국 챌린지 중
🇰🇷 한국 출발 → 맛집·패션·쇼핑으로 세계 일주
나만의 버킷리스트는 mytripfy에서 ✈️
👇
```

   - **링크:** MyTripfy 사이트 URL (예: `https://mytripfy.com` 또는 실제 도메인)

5. **계정 유형:** 설정 → 계정 → **프로페셔널 계정** → **크리에이터** 선택 (인사이트·링크 스티커 사용 가능)

---

### 2-2. 이든 계정

1. **또 다른 이메일**로 새 인스타그램 계정 가입 (이든 전용)
2. **사용자 이름** 예시: `ethan.100countries` / `ethan_mytripfy` / `explorewithethan`
3. **프로필 설정**
   - **프로필 사진:** 이든 대표 이미지 (예: `ethan_late20s_model.png`)
   - **이름:** `Ethan | 100 Countries`
   - **소개문:** 아래 복사해서 사용

```
🌍 100 countries in 1 year
🇺🇸 Starting from the US → adventure & friends worldwide
Start your bucket list on mytripfy 👇
```

   - **링크:** MyTripfy 사이트 URL

4. **프로페셔널 계정** → **크리에이터** 선택

---

## 3. 페이스북 페이지 만들기

각 인스타 계정에 연결할 **페이스북 페이지**를 하나씩 만듭니다.

### 3-1. 수아용 페이스북 페이지

1. [facebook.com/pages/create](https://www.facebook.com/pages/create) 접속 (또는 페이스북 메뉴 → 페이지 만들기)
2. **페이지 이름:** `Sua | 100 Countries Challenge`
3. **카테고리:** `여행 블로거` 또는 `공인/창작자`
4. **소개:** “1년에 100개국 챌린지 · 한국 출발 · 맛집·패션·쇼핑 여행”
5. **프로필/커버 사진:** 수아 이미지로 업로드

### 3-2. 이든용 페이스북 페이지

1. 같은 방식으로 **새 페이지** 생성
2. **페이지 이름:** `Ethan | 100 Countries Challenge`
3. **카테고리:** `여행 블로거` 또는 `공인/창작자`
4. **소개:** “100 countries in 1 year · Starting from the US · Adventure & friends worldwide”
5. **프로필/커버 사진:** 이든 이미지로 업로드

---

## 4. 인스타그램 ↔ 페이스북 연동 (자동 공유)

1. **인스타그램** 앱 또는 웹 → **설정** → **계정**
2. **기타 도구** 또는 **공유 설정** → **Facebook 등 다른 앱에 공유**
3. **Facebook** 선택 → 로그인 후 **수아 인스타**에는 **수아용 페이스북 페이지**, **이든 인스타**에는 **이든용 페이스북 페이지** 연결
4. “**게시물을 Facebook에 공유**” / “**스토리를 Facebook에 공유**” **켜기**

이후 인스타에 올리는 게시물·스토리가 선택한 페이스북 페이지에 자동으로 올라갑니다.

---

## 5. 오늘(Day 1) 첫 게시글 예시

캠페인 **시작일 = 오늘**이므로, 오늘은 **Day 1**입니다.

- **수아:** 오늘 한국에서 1/100 시작  
- **이든:** 오늘 미국에서 1/100 시작  

### 수아 – 첫 게시 (한국)

**캡션 예시 (한글):**
```
🇰🇷 100개국 챌린지, 오늘부터 시작!
한국에서 출발해서 1년 동안 100개국 갈 거예요 ✈️
맛집·패션·쇼핑으로 매일 다른 나라 소개할게요
나만의 버킷리스트는 mytripfy에서 만나요 👇
#mytripfy #100CountriesChallenge #한국 #챌린지시작 1/100
```

### 이든 – 첫 게시 (미국)

**캡션 예시 (영어):**
```
🇺🇸 Day 1 – Starting the 100 countries challenge from the US!
One year, 100 countries. Adventure, new friends, and the best moments. Let’s go 🌍
Start your bucket list on mytripfy 👇
#mytripfy #100CountriesChallenge #USA #ChallengeStart 1/100
```

**이미지:** 하루 **1포스트에 이미지 4장**을 캐러셀(앨범)으로 올립니다.  
첫 게시용 4장은 수아/이든 대표 사진을 한국·미국 배경으로 쓰거나, `node scripts/generate-sns-daily.mjs` 실행 후 나오는 **이미지 1~4번** 프롬프트로 생성한 뒤 **순서대로** 캐러셀에 넣으면 됩니다.

---

## 6. 매일 올리는 방법 (자동/반자동)

- **캡션·프롬프트 자동 생성:** 매일 `node scripts/generate-sns-daily.mjs` 실행 → 수아/이든 각 **캡션 1개 + 이미지 프롬프트 4개** 확인.
- **이미지:** 프롬프트로 AI(Midjourney, DALL·E 등)에서 4장 생성 → 인스타에서 **새 게시물 → 사진 4장 선택 → 캐러셀**로 업로드 후 캡션 붙여넣기.
- **예약 발행:** 4장+캡션을 준비한 뒤 **Meta Business Suite**(business.facebook.com) 또는 **Later**, **Buffer**에서 날짜·시간 지정하면 그 시간에 자동 게시됩니다. (완전 자동으로 “이미지 생성까지” 하려면 Meta API + 이미지 API + 서버 구성이 필요합니다. 자세한 내용은 `docs/sns-campaign-concept.md` §7 참고.)
