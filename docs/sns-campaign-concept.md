# MyTripfy SNS 캠페인 – 100 Countries Challenge

## 1. 캠페인 개요

- **목표:** MyTripfy 사이트와 **100 Countries Challenge** 홍보
- **채널:** Instagram + Facebook (인스타 게시 → 페이스북 자동 공유)
- **기간:** 1년 (365일)
- **콘텐츠:** 매일 **1개 게시글** (캡션 1개 + **이미지 4장** 캐러셀), 수아·이든 각각 독립 계정
- **시작일:** 오늘부터 Day 1
- **출발:** 수아 = 한국(KR), 이든 = 미국(US) — 각각 다른 나라에서 출발

## 3. 스토리라인 – 만남 (6개월마다)

- **1차 만남 (약 6개월 차):** 중앙아시아 또는 동유럽 등 한 도시에서 우연히 마주침 → 스토리/릴스로 “우연한 만남” 연출
- **2차 만남 (12개월 차):** 100번째 국가에서 둘 다 챌린지 완료 → 함께 축하하는 콘텐츠 (공동 게시 또는 각자 게시에서 상대 언급)

---

## 4. SNS 연동 (인스타 → 페이스북)

1. **Meta 계정 센터** (business.facebook.com 또는 인스타 설정)
2. **계정 연결:** Instagram 계정 ↔ Facebook 페이지 연결
3. **자동 공유 설정:**  
   Instagram 설정 → **계정** → **기타 도구** → **Facebook 등 다른 앱에 공유**  
   → “Facebook에 스토리/게시물 공유” 활성화

이렇게 하면 인스타에 올리는 사진·캡션이 선택한 페이스북 페이지에도 자동 게시됩니다.

---

## 5. 일일 콘텐츠 자동 생성 사용법

프로젝트에 포함된 스크립트로 **오늘(또는 지정한 날짜)의 국가**와 **수아/이든 각 “캐러셀 1포스트용” 캡션 1개 + 이미지 프롬프트 4개**를 생성할 수 있습니다.

```bash
# 오늘 = Day 1 (캠페인 시작일 기본값: 오늘)
node scripts/generate-sns-daily.mjs

# 다른 날짜의 콘텐츠
node scripts/generate-sns-daily.mjs --date=2026-03-15

# 캠페인 시작일을 과거로 두고 싶을 때
node scripts/generate-sns-daily.mjs --start=2026-03-14 --date=2026-03-14
```

생성된 프롬프트는 AI 이미지 생성 도구(Midjourney, DALL·E 등)에 붙여 넣어 사용하고, 캡션은 그대로 또는 수정해 인스타/페이스북에 올리면 됩니다.

---

## 6. 포스팅 방식

- **하루 1번** 게시 (수아·이든 각 1포스트/일).
- **한 게시글에 이미지 4장** → 인스타그램 **캐러셀(앨범)**으로 올리기 (순서: 오전 → 점심 → 오후 → 저녁).
- 포스팅 시간은 자유 (예: 저녁 20:00~21:00 KST 권장).

---

## 7. 매일 자동 등록 가능 여부

**완전 자동(사람 개입 없이)**  
- 인스타/페이스북은 “만든 글·이미지를 대신 올려 주는 것”만 허용합니다. **이미지 생성 + 캡션 생성 + 업로드**까지 전부 자동으로 하려면:
  - **Meta 개발자 앱** + **Instagram Graph API / Facebook Graph API**로 게시물 업로드
  - **이미지**: AI 이미지 API(예: DALL·E, Midjourney API 등)로 4장 생성 후 저장
  - **캡션**: 이 스크립트를 서버에서 날짜별로 돌려 텍스트 생성
  - **스케줄**: 서버(cron 등)에서 매일 지정 시간에 위 API로 포스트 발행  
→ 기술적으로 가능하지만, Meta 앱 심사·토큰 관리·이미지 API 비용·서버 운영이 필요합니다.

**반자동(권장)**  
- **콘텐츠만 자동 생성:** `node scripts/generate-sns-daily.mjs`로 매일(또는 `--days=7`로 일주일치) **캡션 + 이미지 프롬프트 4개** 뽑기 → AI로 4장 생성 후 **직접** 인스타에 캐러셀 업로드.
- **예약 발행:** 이미지 4장과 캡션을 준비한 뒤 **Meta Business Suite**(business.facebook.com) 또는 **Later**, **Buffer** 같은 스케줄러에 올려 두면 **지정한 날·시간에 자동 게시**됩니다. (이미지·캡션은 직접 넣어야 함.)
- **상세 절차:** [반자동 워크플로우](sns-workflow-semauto.md) 참고.

정리: **글과 사진을 매일 “완전 자동”으로 등록하려면** Meta API + 이미지 생성 파이프라인 + 서버가 필요하고, **“캡션·프롬프트만 자동 + 올리기는 예약”**이라면 스크립트 + 스케줄러 조합으로 매일 한 번씩 등록하는 방식이 현실적입니다.
