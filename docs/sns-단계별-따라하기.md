# MyTripfy SNS 캠페인 – 단계별 따라하기

처음부터 순서대로 따라 하시면 됩니다. ✓ 표시된 항목을 하나씩 완료해 보세요.

---

## 1단계: 준비물 확인

- [ ] **이메일 2개** – 수아용 1개, 이든용 1개 (인스타그램 가입용)
- [ ] **수아·이든 대표 사진** – 프로필용 (이미 만들어 두었다면 `assets` 폴더 또는 Cursor 프로젝트 `assets` 참고)
- [ ] **Node.js** – 컴퓨터에 설치되어 있어야 스크립트 실행 가능 (설치: [nodejs.org](https://nodejs.org))
- [ ] **OpenAI API 키** (선택) – 이미지 자동 생성하려면 필요 ([platform.openai.com](https://platform.openai.com) → API keys에서 발급)

---

## 2단계: 인스타그램 계정 2개 만들기

### 2-1. 수아 계정 (한국 출발)

1. [instagram.com](https://www.instagram.com) 접속 또는 앱 실행
2. **가입** → 수아 전용 이메일·비밀번호 입력
3. **사용자 이름** 정하기 (예: `sua.100countries`)
4. **프로필 수정**:
   - 프로필 사진: 수아 이미지
   - 이름: `Sua | 100 Countries`
   - 소개:  
     `✨ 1년에 100개국 챌린지 중 🇰🇷 한국 출발 → 맛집·패션·쇼핑으로 세계 일주 나만의 버킷리스트는 mytripfy에서 ✈️ 👇`
   - 링크: MyTripfy 사이트 주소
5. **설정** → 계정 → **프로페셔널 계정** → **크리에이터** 선택

### 2-2. 이든 계정 (미국 출발)

1. **다른 이메일**로 새로 가입
2. 사용자 이름 예: `ethan.100countries`
3. 프로필: 이든 이미지, 이름 `Ethan | 100 Countries`, 소개·링크 입력 (영어 소개는 `docs/sns-account-setup.md` 참고)
4. 프로페셔널 계정 → 크리에이터 선택

---

## 3단계: 페이스북 페이지 2개 만들기

1. [facebook.com/pages/create](https://www.facebook.com/pages/create) 접속
2. **수아용 페이지** 만들기: 이름 `Sua | 100 Countries Challenge`, 카테고리 `여행 블로거`
3. **이든용 페이지** 만들기: 이름 `Ethan | 100 Countries Challenge`, 카테고리 `여행 블로거`
4. 인스타그램 **설정** → **계정** → **Facebook 등 다른 앱에 공유**에서 각 인스타 계정을 해당 페이스북 페이지와 연결하고 **게시물 자동 공유** 켜기

---

## 4단계: OpenAI API 키 설정 (이미지 자동 생성할 때만)

이미지를 스크립트가 자동으로 만들게 하려면 아래 중 **한 가지**만 하면 됩니다.

**방법 A – 배치 파일에서만 쓰기**
1. `scripts` 폴더 안에 `generate-sns-week.bat` 파일을 메모장으로 연다.
2. 맨 위 `@echo off` 다음 줄에 다음 한 줄을 추가한다 (키는 본인 키로 바꾼다):
   ```bat
   set OPENAI_API_KEY=sk-여기에본인API키
   ```
3. 저장 후 배치 파일 실행 시에만 이 키가 사용된다.

**방법 B – 컴퓨터 전체에서 쓰기**
1. Windows 검색에서 **환경 변수** 또는 **시스템 환경 변수 편집** 검색
2. **환경 변수** 버튼 클릭 → **사용자 변수**에서 **새로 만들기**
3. 변수 이름: `OPENAI_API_KEY`, 변수 값: `sk-...` (본인 API 키)
4. 확인 후 터미널/배치를 **다시 열어서** 실행

> API 키가 없어도 됩니다. 그 경우 **게시글(txt)만** 저장되고, 이미지는 직접 AI 도구로 만든 뒤 같은 날짜 폴더에 넣으면 됩니다.

---

## 5단계: 콘텐츠 생성 실행 (일주일치)

1. **프로젝트 폴더**를 연다:  
   `c:\Users\swbae\Documents\mytripfy`
2. 그 안의 **`scripts`** 폴더로 간다.
3. **`generate-sns-week.bat`** 파일을 **더블클릭**한다.
4. 검은 창(명령 프롬프트)이 뜨면서 Day 1 ~ Day 7 콘텐츠가 차례로 출력된다.
5. 끝나면 **아무 키나 누르면** 창이 닫힌다.

**저장 위치 확인**
- `mytripfy\scripts\out\` 폴더를 연다.
- 날짜 폴더가 7개 생긴다: `2026-03-14`, `2026-03-15`, … (실제 날짜는 실행한 날 기준)
- 각 날짜 폴더 안을 보면:
  - `sua.txt`, `ethan.txt` → 게시글(캡션) + 이미지 프롬프트
  - (API 키를 넣었다면) `sua-1.png` ~ `sua-4.png`, `ethan-1.png` ~ `ethan-4.png` → 실제 이미지

---

## 6단계: 이미지가 없을 때 (API 키 없이 실행한 경우)

1. `scripts\out\2026-03-14` 같은 **날짜 폴더**를 연다.
2. **`sua.txt`**를 연다. 맨 아래 **[이미지 4장 순서대로 생성]** 안에 영어 문장 4개가 있다.
3. 그 4개 문장을 **1번, 2번, 3번, 4번 순서대로** Midjourney·DALL·E 등에 넣어 이미지 4장을 만든다.
4. 만든 4장을 **같은 날짜 폴더**에 `sua-1.png`, `sua-2.png`, `sua-3.png`, `sua-4.png` 이름으로 저장한다.
5. **`ethan.txt`**도 같은 방식으로 4장 만들고 `ethan-1.png` ~ `ethan-4.png`로 저장한다.

---

## 7단계: 인스타그램에 게시 (수아·이든 각각)

**수아 계정으로**
1. 인스타그램 앱 또는 웹에서 **수아 계정**으로 로그인
2. **새 게시물** → **여러 장 선택** → 해당 날짜 폴더에서 `sua-1.png` ~ `sua-4.png` 순서대로 선택
3. **캡션**에는 `sua.txt` 파일을 열어 **[캡션 · 한 포스트에 그대로 사용]** 아래 문장을 **복사해 붙여넣기**
4. **공유** 클릭

**이든 계정으로**
1. **이든 계정**으로 로그인
2. 새 게시물 → `ethan-1.png` ~ `ethan-4.png` 선택
3. 캡션은 `ethan.txt`의 **[Caption · use as single post]** 아래 문장 복사해 붙여넣기
4. 공유

(인스타–페이스북 연동을 했다면 페이스북에도 자동으로 올라갑니다.)

---

## 8단계: 예약 발행으로 미리 올리기 (선택)

1. [business.facebook.com](https://business.facebook.com) 접속 → **Meta Business Suite** 로그인
2. 왼쪽에서 **인스타그램** 선택 → **콘텐츠** → **게시물 예약** 또는 **만들기**
3. **수아 계정** 선택 → 이미지 4장 업로드(sua-1~4 순서) → 캡션 붙여넣기 → **게시 시간**에서 날짜·시간 선택 → 예약
4. **이든 계정**도 같은 방식으로 다른 날짜 폴더 이미지로 예약
